### 1. 개요
- 개인프로젝트 약정보 웹애플리케이션 yakssok의 사용자 홈페이지입니다.  
- Vue CLI를 활용하여 SPA로 만들었으며 REST API로 서버에서 데이터를 가져오는 구조로 설계하였습니다.

***

### 2. 사용기술
  - Vue CLI
  - Azure Cloud Blob Storage
